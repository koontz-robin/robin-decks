## 📌 How to Add a New Deck

When Robin creates a new deck or dashboard:
1. Save the `.html` file to `/home/openclaw/.openclaw/workspace/`
2. Add it to the appropriate section above
3. Push to `robin-decks` repo so it's live on GitHub Pages
4. Note the live URL here

**Do not use** `htmlpreview.github.io` or `raw.githack.com` — these are blocked by enterprise email security (Proofpoint). Always use `https://koontz-robin.github.io/robin-decks/` as the base.
