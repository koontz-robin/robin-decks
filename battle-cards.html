The 800-lb gorilla of the MSP PSA market. Comprehensive platform with deep ecosystem of integrations. Acquired by Thoma Bravo (PE) in 2019. Known for breadth but also complexity, aggressive contracts, and slow innovation.

Strengths vs. Weaknesses

Strengths

  • Largest MSP ecosystem — most 3rd-party integrations
  • Deep functionality — covers nearly every MSP workflow
  • Strong community (IT Nation, peer groups)
  • Mature product with long track record
  • ConnectWise Sell provides quoting/CPQ
  • Large partner network and training resources

Weaknesses

  • Aggressive lock-in contracts (multi-year, auto-renew, hard to exit)
  • Bloated and complex UI — steep learning curve, feels outdated
  • Billing module is basic — no native usage-based or telecom billing
  • Over-promises on new features (Asio/Sidekick AI still in preview)
  • No integrated payment processing — requires third-party
  • Security concerns (ScreenConnect vulnerability 2024)
  • Pricing has increased significantly; nickel-and-dime add-on model
  • Project billing broken — labor lines convert to generic "fixed fee," stripping scope/description. Manual copy-paste every invoice.
  • Project conversion bug (unfixed since onboarding) — estimated labor time/cost never auto-populates on finance tab. Manual calculator math required.
  • Agreement invoicing is 100% manual — CW recommends against auto-generation. No way to set generation date separate from billing period start. Costs admins ~1 full day/month.
  • Quote data doesn't flow to invoices — quote reference #, scope notes, and labor line descriptions must be manually re-entered every time.
